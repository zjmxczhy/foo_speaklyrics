# 朗读 LRC 歌词组件更新日志

## 0.5.3-2026-07-09

### 调整：更换 Tolk 运行库
更换随组件打包的 Tolk 运行库，改用 boz700908/tolk 发布版，继续通过 Tolk 调用读屏接口朗读歌词。

### 调整：更新保益读屏支持 DLL
同步替换保益读屏相关 DLL。新的运行库使用 `byctrl.dll`、`byctrl-x64.dll` 和 `byctrl.conf`，不再打包旧的 `BoyCtrl.dll`、`BoyCtrl-x64.dll`。

x86、x64 和通用组件安装包都会随包带上新的 Tolk 与保益支持文件；打包脚本也已调整为自动包含 Tolk 目录下的 `.dll`、`.ini`、`.conf` 文件，避免后续更换运行库时遗漏依赖文件。

发现新版的tolk的问题可以在这个仓库和作者反应
Tolk 来源仓库：https://github.com/boz700908/tolk

### 调整：自动朗读关闭时仍会加载和下载歌词
调整“自动朗读”的职责边界：现在“自动朗读”只控制是否把歌词朗读出来，不再阻止歌词加载、后台扫描和自动下载。

当用户关闭自动朗读时，如果已经勾选歌词下载来源，组件仍会在没有找到本地 LRC 时调用下载器下载歌词，并继续按后台扫描时间加载下载完成的歌词。这样即使不启用自动朗读，也可以使用复制歌词、按歌词跳转等依赖已加载 LRC 的功能。

如果不希望组件联网下载歌词，可以在“朗读歌词设置 -> LRC歌词 -> 歌词下载来源”中取消全部来源；全部不选时，组件只扫描本地歌曲同目录、设置的 LRC 歌词目录和 LRC 歌词临时目录。

### 新增：临时 LRC 清单清理机制
新增 `foo_speaklyrics_temp_lrc_manifest.txt` 临时歌词清单。组件把自动下载到 LRC 歌词临时目录的歌词路径记录到清单中，并在 foobar2000 启动或退出时尝试清理这些临时 LRC。

清单文件保存在 foobar2000 配置目录中，路径由 foobar2000 SDK 自动获取，兼容便携版、安装版、foobar2000 1.5/1.6 x86 和 foobar2000 2.x。

只有下载到“LRC歌词临时目录”的歌词会写入清单；下载到用户设置的正式 LRC 歌词目录、手动加载的歌词、手动放入临时目录但没有写入清单的歌词，都不会被这个清理机制删除。

### 优化：复制歌词功能支持自定义格式
将“复制无时间戳歌词”调整为“复制歌词”，复制方式不再固定为无时间戳文本。现在可以按用户选择复制，也可以在设置中固定为复制时间戳歌词、复制无时间戳歌词或复制分行歌词。

在“朗读歌词设置 -> LRC歌词”中增加“复制歌词默认方式”组合框和“设为默认(&F)”按钮。用户可以把常用复制方式保存为默认值，后续点击“复制歌词”时直接按默认方式处理。

同时增加“歌词分行符号”编辑框。用户可以输入用于分行的标点或字符，例如逗号、句号、问号等。默认留空；留空时“复制分行歌词”的结果与“复制无时间戳歌词”一致，不额外拆分歌词内容。

### 新增：手动播报当前歌曲信息
在“视图 -> 朗读歌词”菜单中新增“查看当前播放歌曲信息”功能，也可以在 foobar2000 中为该菜单命令自定义全局快捷键。

当用户在听歌过程中需要确认当前曲目信息时，可以随时手动触发播报。播报内容会复用“朗读歌词设置 -> 常规设置”里的“切换歌曲播报内容”组合框，例如标题和艺术家、标题、艺术家和标题、文件名或文件名不含扩展名。

该功能不受“切换歌曲时播报歌曲信息”复选框是否启用影响；即使关闭自动切歌播报，也可以手动查看当前歌曲信息。

### 调整：移除启动后隐藏主窗口功能
“启动后隐藏 foobar2000 主窗口”功能已迁移到 `foobar2000 无障碍增强` 组件中，由新组件在 foobar2000 参数选项里统一设置和执行。

朗读歌词组件不再显示“启动后隐藏主窗口”复选框和延迟毫秒设置，也不再在启动时处理主窗口隐藏，让本组件专注于 LRC 歌词朗读。

理论上原配置 GUID 已由 `foobar2000 无障碍增强` 继续使用，因此已勾选过该功能的用户安装新组件后可以沿用原来的设置值。

## 0.5.2-2026-06-30

### 新增：日志和故障排查
新增组件日志文件，用于记录歌词加载、下载器启动、下载失败、Tolk 加载失败、SAPI5.1 朗读失败、启动隐藏主窗口等关键诊断信息。  
日志文件保存在 foobar2000 配置目录下，文件名为 `foo_speaklyrics.log`。  
当日志超过约 2MB 时，组件会自动轮换为 `foo_speaklyrics.log.old`。

### 调整：关键错误同时写入 foobar2000 控制台
关键诊断信息会同时写入 foobar2000 控制台和日志文件。  
用户反馈“无法朗读”“无法下载”“找不到歌词”“安装包启动被隐藏”等问题时，可以直接提供 `foo_speaklyrics.log` 方便排查。

## 0.5.1-2026-06-30

### 优化：最小化的幽灵窗口
优化“启动后隐藏 foobar2000 主窗口”功能在部分托盘模式下的处理逻辑。  
当 foobar2000 设置为关闭时隐藏到系统托盘、并启用启动后延迟隐藏时，组件现在会在调用 foobar2000 隐藏主窗口命令后，再对主窗口执行隐藏兜底处理，减少 Alt+Tab 中仍能切换到 foobar2000 窗口的情况。

### 调整：启动隐藏后短时间复查窗口状态
启动隐藏执行后，组件会在短时间内继续复查主窗口状态。  
如果 foobar2000 或界面模块在启动过程中又短暂恢复了主窗口，组件会再次隐藏，提升不同最小化/托盘设置下的一致性。
如果用户在这段时间内通过热键主动显示并激活 foobar2000 主窗口，组件会停止后续复查隐藏，避免刚显示出来又被自动隐藏。

### 修复：安装组件时不触发启动隐藏
修复在 foobar2000 未启动时直接打开 `.fb2k-component` 安装包，组件的启动隐藏逻辑可能把安装流程中的 foobar2000 窗口隐藏，导致需要重新打开安装包的问题。  
现在检测到本次启动来自组件安装包时，会跳过启动后隐藏主窗口。

## 0.5.0-2026-06-29

### 修复：x86 组件无法在 foobar2000 1.5/1.6 中使用
修复 x86 组件包安装到 foobar2000 1.5/1.6 后，启动时可能因为读取新版配置接口而崩溃的问题。  
现在 x86 组件改为使用 foobar2000 SDK 的兼容配置写法，在 foobar2000 1.5/1.6 x86 和 foobar2000 2.x x86 中都可以使用。
注：由于对旧版配置做兼容，重新安装可能需要重新设置参数。

### 修复：foobar2000 首次启动托盘模式下的幽灵窗口
修复在 foobar2000 设置为最小化到托盘、全程不显示主窗口时，首次启动后读屏仍可能读到“艺术家 - [歌曲名] 专辑名 [foobar2000]”标题的问题。
现在组件会在 foobar2000 主窗口初始化完成后，按设置延迟调用 foobar2000 的隐藏主窗口命令，减少首次启动时残留在 Alt+Tab 或读屏窗口列表中的幽灵窗口。

### 新增：启动后隐藏主窗口开关
在“朗读歌词设置 -> 常规设置”中新增“启动后隐藏主窗口”复选框。  
默认未启用。有需要的用户可以自行勾选，启用后组件会在 foobar2000 启动后自动隐藏主窗口。

### 新增：启动后隐藏延迟毫秒
在“朗读歌词设置 -> 常规设置”中新增“启动后隐藏延迟毫秒”编辑框。  
默认值为 `800`，表示启动后延迟 0.8 秒执行隐藏主窗口。  
如果某些机器上首次启动仍能短暂读到幽灵窗口，可以尝试设置为 `1500` 或 `2000`。

## 0.4.1-2026-06-26

### 修复：SAPI5.1 切换歌曲播报会排队朗读
修复在使用 SAPI5.1 语音时，切换歌曲播报标题、艺术家等歌曲信息可能排队到上一段朗读结束后才播放的问题。
现在切换歌曲播报会打断当前 SAPI5.1 朗读，并清空 SAPI5.1 当前朗读队列，让新歌曲信息优先播报。

## 0.4.0-2026-06-25

### 新增：切换歌曲时播报歌曲信息，感谢雪落孤草的建议。
在“朗读歌词设置 -> 常规设置”中新增“切换歌曲时播报歌曲信息”复选框。  
启用后，切换到新歌曲时会通过当前选择的语音输出方式播报歌曲信息。

### 新增：切换歌曲播报内容
新增“切换歌曲播报内容”组合框，支持选择以下播报方式：
1. 标题和艺术家。
2. 标题。
3. 艺术家和标题。
4. 文件名。
5. 文件名不含扩展名。

如果歌曲缺少标题或艺术家信息，组件会自动回退到文件名或 foobar2000 路径信息。

### 新增：切换歌曲播报延迟毫秒
新增“切换歌曲播报延迟毫秒”编辑框。  
默认值为 `0`，表示切歌后立即播报。  
可以设置为 `300`、`1000` 等数值，让组件在切歌后延迟播报歌曲信息。

### 调整：避免切歌播报和开头歌词冲突
启用切歌播报后，组件会优先播报歌曲信息。  
如果设置了延迟播报，在歌曲信息播报前不会抢先朗读开头歌词，减少切歌瞬间语音互相打断的问题。

## 0.3.0-2026-06-23

### 新增：添加当前时间歌词
在“视图 -> 朗读歌词”菜单中新增“添加当前时间歌词”功能。  
该功能用于手动制作或补全 LRC 歌词，打开窗口后会自动读取当前播放位置并填入 LRC 时间戳。

窗口包含“当前时间”“当前时间歌词”“编码”“保存”“另存为”“打开”“预览”“关闭”等控件。  
在“当前时间歌词”编辑框中输入歌词后按 Enter，会把当前时间和歌词追加到预览内容中，并自动刷新当前播放时间。

### 新增：刷新当前时间
在“添加当前时间歌词”窗口中新增“刷新当前时间(T)”按钮，快捷键为 Alt+T。  
点击后会重新读取当前播放位置，并刷新“当前时间”编辑框，不会改变播放或暂停状态。

### 新增：LRC 编辑保存能力
新增 LRC 文本打开、保存和另存为流程。  
打开已有 LRC 时会自动识别编码，保存时会按窗口中选择的编码写入文件。

### 调整：编码选项补充 UTF-8 with BOM
LRC 编码列表新增“Unicode (UTF-8 with BOM)”选项。  
新建歌词默认使用 Unicode UTF-8；如果需要兼容某些旧软件，可手动选择 UTF-8 with BOM。

### 说明：保存后不自动绑定当前歌词
“添加当前时间歌词”保存后不会自动绑定为当前朗读歌词。  
如果要让组件自动识别保存后的歌词，建议保存到歌曲同目录、设置的 LRC 歌词目录，或手动加载保存后的 LRC 文件。

## 0.2.0-2026-06-19

### 新增：TTS 语音设置
在“朗读歌词设置”窗口中新增“TTS语音”标签页。

### 新增：SAPI5.1 语音朗读
TTS 语音页中默认勾选“使用屏幕阅读器”。  
勾选时继续使用 Tolk 调用屏幕阅读器朗读歌词。  
取消勾选后显示 SAPI5.1 相关设置，可选择 Windows 已安装的 SAPI5.1 语音，并可调整语速和试听。

### 新增：本地歌曲同目录 LRC 自动识别
播放本地歌曲时，组件会优先在歌曲文件所在目录查找同名 LRC 歌词。

### 新增：下载到指定的歌词目录
在“朗读歌词设置”的“LRC歌词”标签页中新增“下载到指定的歌词目录”复选框。  
默认未选中，歌词自动下载后保存到 LRC 歌词临时目录。  
选中后，歌词自动下载和搜索歌词自动匹配下载会保存到用户设置的 LRC 歌词目录，适合长期保存歌词文件。

### 调整：LRC 查找顺序
当前自动查找顺序为：
1. 手动导入且绑定到当前歌曲的 LRC。
2. 本地歌曲同目录下的同文件名 LRC，例如 `青花瓷.lrc`。
3. 本地歌曲同目录下的 `艺术家 - 歌名.lrc`。
4. 本地歌曲同目录下的 `歌名 - 艺术家.lrc`。
5. 本地歌曲同目录下按歌名、艺术家做模糊匹配。
6. 用户设置的 LRC 歌词目录。
7. LRC 歌词临时目录。
8. 如果仍未找到，并且启用了歌词来源，则调用下载器从网络来源下载。

### 说明：网络歌曲不扫描同目录
网络来源的歌曲没有本地文件目录，因此不会执行“同目录 LRC 查找”。
网络歌曲会跳过同目录判断，继续检查设置的 LRC 歌词目录、临时目录，然后再按设置的来源下载。

### 调整：设置窗口布局
在“朗读歌词设置”窗口新增“常规设置”标签页。  
当前标签页顺序调整为：`常规设置`、`LRC歌词`、`TTS语音`。

以下控件已移动到“常规设置”标签页：
1. 自动朗读。
2. LRC歌词编码。
3. 时间偏移（毫秒，负数提前）。
4. 无lrc歌词后台扫描时间毫秒。
5. 歌词朗读有效时间毫秒。

### 调整：LRC歌词标签页焦点顺序
“LRC歌词”标签页的焦点顺序调整为：
1. 歌词下载来源。
2. 下载到指定的歌词目录。
3. LRC文件。
4. 浏览文件。
5. LRC歌词目录。
6. 浏览文件夹。
7. LRC歌词临时目录。
8. 浏览文件夹。
9. 切换歌曲删除临时lrc时间毫秒。
