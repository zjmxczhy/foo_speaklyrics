# 朗读 LRC 歌词组件使用方法

组件名称：**朗读 LRC 歌词**  
组件文件名：`foo_speaklyrics.dll`  
适用软件：foobar2000 2.1 及以上。
主要用途：在播放歌曲时读取 LRC 歌词，并通过屏幕阅读器/Tolk 或 SAPI5.1 朗读当前歌词。

---

## 1. 组件主要功能

本组件可以实现以下功能：

1. 加载本地 `.lrc` 歌词文件。
2. 设置固定的 LRC 歌词目录。
3. 设置临时 LRC 歌词目录。
4. 自动扫描歌词并朗读。
5. 调用外部歌词下载器下载歌词。
6. 支持选择歌词下载来源。
7. 支持搜索 LRC 歌词并下载到临时目录。
8. 支持复制无时间戳歌词。
9. 支持按歌词跳转播放位置。
10. 支持添加当前时间歌词，用于手动制作或补全 LRC 歌词。
11. 支持通过 Tolk 调用屏幕阅读器朗读歌词。
12. 支持在没有使用屏幕阅读器时改用 SAPI5.1 语音朗读。

## 2. 安装方法

### 2.1 x64 版 foobar2000
如果你使用的是 64 位 foobar2000便携版，组件目录通常是：
Foobar2000\user-components-x64\foo_speaklyrics

### 2.2 x86 版 foobar2000
如果你使用的是 32 位 foobar2000便携版，组件目录通常是：
foobar2000\user-components\foo_speaklyrics

### 2.3 使用组件安装包安装
foo_speaklyrics-x64.fb2k-component
foo_speaklyrics-x86.fb2k-component
可以在 foobar2000 里打开：
文件 -> 参数选项 -> 组件
然后选择安装组件包。
安装完成后需要重启 foobar2000。

## 4. 朗读歌词设置

打开：
视图 -> 朗读歌词 -> 朗读歌词设置

设置窗口分为四个选项卡：
1. 常规设置
2. LRC歌词
3. TTS语音
4. 关于

切换选项卡后，焦点会保留在选项卡控件上。按 Tab 可以进入当前选项卡里的控件。

---

### 4.1 常规设置

常规设置里放的是朗读、扫描和切换歌曲播报相关的基础选项。

#### 4.1.1 自动朗读

用于控制是否自动朗读歌词。
- 选中：播放歌曲时自动朗读 LRC 歌词。
- 未选中：不自动朗读，需要手动开启。

默认是关闭。
开启时会播报：
LRC朗读已打开
关闭时会播报：
LRC朗读已关闭

#### 4.1.2 LRC 歌词编码

用于选择 LRC 歌词文件的文本编码。
默认值：
自动

一般情况下建议保持“自动”，组件会自动判断常见歌词编码。
如果某个歌词打开后出现乱码、问号、文字不正常，可以手动选择对应编码后再测试。

目前可选编码包括：
1. 自动
2. Unicode UTF-8
3. Unicode UTF-8 with BOM
4. Unicode UTF-16 LE
5. Unicode UTF-16 BE
6. Simplified Chinese GBK
7. Chinese Simplified GB18030
8. Traditional Chinese Big5
9. ANSI/OEM Japanese Shift-JIS
10. ANSI/OEM Korean
11. Western European Windows-1252
12. Turkish Windows-1254
13. 系统默认 ANSI

说明：
- 带 BOM 的 UTF-8、UTF-16 LE、UTF-16 BE 歌词会优先自动识别。
- 组件下载器下载的歌词通常是 UTF-8，一般不需要手动修改。
- 国内常见旧 LRC 歌词如果乱码，可以优先尝试 GBK 或 GB18030。

#### 4.1.3 设定朗读器前置时间毫秒

用于控制歌词朗读的时间偏移。
单位：毫秒。

示例：
0
表示不提前也不延后。

-1000
表示提前 1 秒朗读。

1000
表示延后 1 秒朗读。

注意：如果设置为负数，比如 `-1500`，开头时间很靠前的歌词可能会因为提前后变成负时间，从而出现开头歌词不朗读的情况。遇到开头歌词漏读，可以先把这个值改回 `0` 测试。

#### 4.1.4 无 LRC 歌词后台扫描时间毫秒

用于控制当前没有歌词时，后台多久扫描一次歌词目录或临时目录。

例如：
500
表示每 0.5 秒扫描一次。

3000
表示每 3 秒扫描一次。

如果你的歌词是外部下载器下载到临时目录，建议不要设置太大，否则新歌词下载完成后可能要等比较久才会被发现。

#### 4.1.5 歌词朗读有效时间毫秒

用于控制一行歌词在多长时间内算有效。
默认值：
3000
也就是 3 秒。

这个功能主要用于避免快进、快退时读到已经过期的歌词。
如果设置太大，快进到纯音乐部分时，可能仍然读到前面的歌词。  
如果设置太小，有些歌词可能来不及朗读。

#### 4.1.6 切换歌曲时播报歌曲信息

用于控制切换到新歌曲时，是否先播报歌曲信息。

- 选中：切换歌曲后，通过当前选择的语音输出方式播报歌曲信息。
- 未选中：切换歌曲时不播报歌曲信息，只按原来的歌词朗读逻辑工作。

启用后，组件会优先播报歌曲信息，减少切歌瞬间歌曲信息和开头歌词互相打断的问题。

#### 4.1.7 切换歌曲播报内容

用于选择切换歌曲时播报哪些内容。

可选项包括：
1. 标题和艺术家。
2. 标题。
3. 艺术家和标题。
4. 文件名。
5. 文件名不含扩展名。

如果歌曲缺少标题或艺术家信息，组件会自动回退到文件名或 foobar2000 路径信息。

#### 4.1.8 切换歌曲播报延迟毫秒

用于控制切换歌曲后延迟多久再播报歌曲信息。

默认值：
0
表示切歌后立即播报。

示例：
300
表示切歌后延迟 0.3 秒播报。

1000
表示切歌后延迟 1 秒播报。

### 4.2 LRC歌词

LRC歌词选项卡里放的是歌词下载来源、下载保存位置、歌词文件、歌词目录和临时目录相关设置。

焦点顺序是：
1. 歌词下载来源。
2. 下载到指定的歌词目录。
3. LRC 文件。
4. 浏览文件。
5. LRC 歌词目录。
6. 浏览文件夹。
7. LRC 歌词临时目录。
8. 浏览文件夹。
9. 切换歌曲删除临时 LRC 时间毫秒。

#### 4.2.1 歌词下载来源

这里可以勾选自动下载歌词时使用的来源。
如果所有下载来源都没有选中，组件只会走本地歌词扫描，不会调用下载器。

目前可用来源包括：
1. LRCLIB
2. QQ音乐1号源
3. QQ音乐2号源
4. 网易云音乐

实际显示顺序以设置窗口为准。

#### 4.2.2 下载到指定的歌词目录

默认未选中。

- 未选中：自动下载和搜索自动匹配下载的歌词保存到 LRC 歌词临时目录。
- 选中：自动下载和搜索自动匹配下载的歌词保存到 LRC 歌词目录。

如果想长期保存下载到的歌词，可以选中这个复选框。
如果只想临时使用下载歌词，建议保持未选中。

#### 4.2.3 LRC 文件

用于临时手动导入一个 `.lrc` 歌词文件。
这个输入框不会长期记忆手动导入的歌词路径。
选择文件后点确定，会把这个歌词用于当前歌曲。

#### 4.2.4 LRC 歌词目录

这是固定歌词目录。
适合长期保存歌词。
例如：
D:\Lyrics

组件会在这个目录里查找当前歌曲对应的 `.lrc` 文件。
如果选中“下载到指定的歌词目录”，下载器也会把歌词保存到这个目录。

#### 4.2.5 LRC 歌词临时目录

这是临时歌词目录。
适合给下载器使用。
默认值：
%Temp%

也就是当前 Windows 用户的临时目录，通常类似：
C:\Users\你的用户名\AppData\Local\Temp

注意：`%Temp%` 会在组件实际使用时自动展开成真实路径，不会当成普通文件夹名称。
你也可以自己改成其他目录，例如：
D:\Temp

组件可以从这个目录读取临时下载的歌词。
切换歌曲时，组件可以按设置删除上一首歌的临时歌词。

#### 4.2.6 切换歌曲删除临时 LRC 时间毫秒

默认值：
0

含义：切换歌曲后立即删除上一首歌的临时歌词。

如果设置为：
1000
表示切换歌曲后延迟 1 秒删除。

这个功能可以避免刚切歌又切回来时重复下载歌词。
注意：停止播放不会删除临时歌词，切换到下一首歌才会删除。

### 4.3 TTS语音

TTS语音选项卡用于选择歌词朗读输出方式。

#### 4.3.1 使用屏幕阅读器

默认选中。

选中时，组件使用 Tolk 调用屏幕阅读器朗读歌词，例如争渡、NVDA 等。  
选中时，不显示 SAPI5.1 的语音选择、语速和试听控件。

如果你主要使用读屏软件，建议保持选中。

#### 4.3.2 使用 SAPI5.1 朗读

取消勾选“使用屏幕阅读器”后，会显示 SAPI5.1 相关设置。

可设置内容包括：
1. 选择语音。
2. 语速滑块。
3. 试听文本。
4. 试听(T) 按钮，Alt+T。

选择语音组合框会列出 Windows 中已经安装的 SAPI5.1 语音。
如果列表为空或只有默认语音，说明系统里可用的 SAPI5.1 语音较少。

语速滑块范围是：
-10 到 10

0 是默认语速。  
负数更慢，正数更快。

试听文本会自动生成类似：
你选择了xxxx语音，语速xx。

点击“试听(T)”会使用当前选择的语音和语速播放试听文本。

#### 4.3.3 何时使用 SAPI5.1

如果没有启动屏幕阅读器，或者你想让歌词通过 Windows 语音库朗读，可以取消勾选“使用屏幕阅读器”，改用 SAPI5.1。

如果使用 SAPI5.1 后出现不稳定、无声或语音异常，可以重新勾选“使用屏幕阅读器”，恢复使用 Tolk/读屏朗读。

### 4.4 关于

关于选项卡用于查看组件用途和相关发布地址。

主要控件包括：
1. 主要用途，只读编辑框。
2. 查看GitHub仓库。
3. 前往GitHub发布页。
4. 备用gitee发布页。

点击按钮会调用系统默认浏览器打开对应页面。

### 4.5 确定、取消和应用

设置窗口底部有三个按钮：
确定(O)，Alt+O
取消(C)，Alt+C
应用(A)，Alt+A

说明：
- 确定：保存设置并关闭窗口。
- 取消：不保存本次修改并关闭窗口。
- 应用：保存设置但不关闭窗口。
- 应用按钮只有在设置内容发生修改后才会显示或可用。

## 5. 加载本地 LRC 歌词

菜单位置：
视图 -> 朗读歌词 -> 加载本地LRC歌词
这个功能用于手动选择一个 `.lrc` 文件。
适合临时测试某个歌词文件。
注意：手动加载的歌词不建议长期绑定到所有歌曲。如果切歌后仍然一直使用手动加载的歌词，需要清除或重新加载当前歌曲对应的歌词。

---

## 6. 设置 LRC 歌词目录

菜单位置：
视图 -> 朗读歌词 -> 设置LRC歌词目录
这个目录用于保存长期使用的歌词。
建议歌词文件命名尽量包含歌名和歌手，例如：
刘珂矣 - 半壶纱.lrc
周杰伦 - 青花瓷.lrc
组件会根据当前歌曲的标题和艺术家信息匹配歌词。

---

## 7. 设置 LRC 歌词临时目录

菜单位置：
视图 -> 朗读歌词 -> 设置LRC歌词临时目录
这个目录适合放外部下载器下载出来的歌词。
默认是：
%Temp%
也就是当前 Windows 用户的临时目录，通常在 C 盘用户目录下。
组件支持环境变量路径，例如：
%Temp%
%Temp%\foo_speaklyrics
%UserProfile%\LyricsTemp
也可以改成固定目录，例如：
D:\Temp
如果组件启用了下载来源，下载器会把歌词保存到这个目录，组件随后扫描并加载。

---

## 8. 歌词下载来源

组件支持选择歌词下载来源。
如果所有下载来源都没有选中，组件只会走本地歌词扫描，不会调用下载器。
目前可用来源包括：
1. LRCLIB
2. QQ音乐1号源
3. QQ音乐2号源
4. 网易云音乐
实际显示顺序以设置窗口为准。

### 8.1 多个来源同时选中会怎样

可以同时选中多个来源。
组件会按顺序尝试下载。
一般逻辑是：
1. 先试第一个来源。
2. 如果没有找到歌词，再试下一个来源。
3. 找到可用歌词后保存到临时目录。

建议常用设置：
QQ音乐1号源
QQ音乐2号源
网易云音乐
如果你只想用本地歌词，把所有来源都取消勾选即可。

## 9. 搜索 LRC 歌词

菜单位置：
视图 -> 朗读歌词 -> 搜索LRC歌词
这个功能用于手动搜索歌词，并把选中的歌词下载到临时歌词目录。
适合自动下载没有匹配到、或者需要用户自己挑选歌词版本的情况。

### 9.1 窗口控件

搜索窗口包含以下控件：
自动填入当前播放信息(A) 按钮，Alt+A
标题 编辑框
艺术家，可选 编辑框
搜索(S) 按钮，Alt+S
结果 列表
关闭(C) 按钮，Alt+C

### 9.2 搜索方法

1. 打开搜索 LRC 歌词窗口。
2. 如果要使用当前播放歌曲的信息，按“自动填入当前播放信息(A)”按钮。
3. 也可以手动在“标题”和“艺术家，可选”编辑框里输入歌曲信息。
4. 在标题或艺术家编辑框里按回车可以直接搜索。
5. 按“搜索(S)”按钮也可以开始搜索。
6. 搜索时会显示进度条，搜索完成后焦点会进入结果列表。
7. 如果没有搜索到结果，结果列表会显示没有搜索到结果。

### 9.3 结果列表

结果列表的显示格式类似：
半壶纱，刘珂矣，来源：QQ音乐1号源

用上下光标选择结果。
按回车会把选中的歌词下载到临时 LRC 歌词目录。
下载完成后不单独提示，因为组件会自动扫描临时目录并朗读歌词。

### 9.4 下载到固定歌词目录

在结果列表中打开上下文菜单，可以选择：
下载到设置的LRC歌词目录
这个功能会把选中的歌词保存到固定 LRC 歌词目录，适合长期保存。

### 9.5 注意事项

- 搜索使用朗读歌词设置里勾选的下载来源。
- 如果所有下载来源都没有勾选，搜索不会使用网络来源。
- 标题必须填写，艺术家可以不填写。
- 如果没有填写艺术家，组件可以参考当前播放歌曲的艺术家信息辅助匹配。
- 搜索窗口不会重复打开；再次点击菜单会把已经打开的窗口拉回前台。
- 可以用 Esc、Alt+F4 或关闭(C) 按钮关闭窗口。

## 10. 复制无时间戳歌词

菜单位置：
视图 -> 朗读歌词 -> 复制无时间戳歌词
这个功能会把当前 LRC 歌词里的时间戳去掉，只复制歌词正文。
例如原歌词：
[00:12.00]青花瓷
[00:15.00]天青色等烟雨
复制后类似：
青花瓷
天青色等烟雨
适合复制歌词文本到其他地方使用。

---

## 11. 按歌词跳转

菜单位置：
视图 -> 朗读歌词 -> 按歌词跳转
打开后会出现一个歌词列表窗口。

### 11.1 使用方法

1. 打开按歌词跳转窗口。
2. 用上下光标选择歌词行。
3. 按回车跳转到这一句歌词对应的播放位置。
4. 如果当前是停止状态，按回车会先开始播放，再跳转。
5. 跳转后窗口不会自动关闭。

### 11.2 鼠标操作

鼠标单击歌词：只选择，不跳转。  
鼠标双击歌词：跳转到对应播放位置。

### 11.3 关闭窗口

可以用以下方式关闭：
Esc
Alt+F4
关闭(C) 按钮，Alt+C

### 11.4 注意事项

- 列表里只显示歌词正文，不显示时间戳。
- 没有时间戳的歌词行不会放进跳转列表。
- 多行相同时间戳的歌词会都列出来。
- 如果当前没有可用歌词，会提示：
当前没有可用歌词
- 如果歌词正在后台下载，窗口可能先显示空列表，下载完成后会自动刷新。
- 切换歌曲后，窗口里的歌词列表会自动刷新成新歌歌词。

---

## 12. 添加当前时间歌词

菜单位置：
视图 -> 朗读歌词 -> 添加当前时间歌词

这个功能用于手动制作或补全 LRC 歌词。打开窗口后，组件会把当前播放位置自动填入为 LRC 时间戳。

### 12.1 窗口控件

添加当前时间歌词窗口包含以下控件：
1. 当前时间 编辑框。
2. 刷新当前时间(T) 按钮，Alt+T。
3. 当前时间歌词 编辑框。
4. 编码 组合框。
5. 保存(S) 按钮，Alt+S。
6. 另存为(A) 按钮，Alt+A。
7. 打开(O) 按钮，Alt+O。
8. 预览 编辑框。
9. 关闭(C) 按钮，Alt+C。

窗口不会重复打开；再次点击菜单会把已经打开的窗口拉回前台。

### 12.2 添加歌词方法

1. 播放歌曲到需要添加歌词的位置。
2. 打开“添加当前时间歌词”窗口。
3. “当前时间”编辑框会自动填入类似 `[01:23.45]` 的 LRC 时间。
4. 如果需要重新读取当前播放位置，可以按“刷新当前时间(T)”按钮，或使用 Alt+T。
5. 在“当前时间歌词”编辑框输入歌词正文。
6. 按 Enter，会把“当前时间 + 歌词”追加到“预览”编辑框。
7. 追加后，“当前时间歌词”会清空，“当前时间”会刷新为当前播放时间。
8. 继续输入下一句歌词。

### 12.3 刷新当前时间

“刷新当前时间(T)”按钮用于重新读取当前 foobar2000 播放位置，并写入“当前时间”编辑框。

使用场景：
1. 打开窗口后歌曲已经继续播放，需要重新取当前时间。
2. 手动修改过当前时间后，想恢复为当前播放位置。
3. 暂停后继续播放，需要重新抓取新的时间点。

### 12.4 打开和继续编辑

点击“打开(O)”可以打开已有 `.lrc` 文件。

打开后：
1. 文件内容会显示在“预览”编辑框。
2. 组件会自动识别 LRC 文件编码。
3. 编码组合框会尽量切换到识别出的编码。
4. 后续点击“保存(S)”会直接保存到当前打开的文件。

如果当前预览内容有未保存修改，打开其他文件或关闭窗口前会提示是否保存。

### 12.5 保存和另存为

保存(S)：
- 如果已经打开或保存过文件，会直接保存到当前文件。
- 如果还没有文件路径，会弹出系统保存对话框。

另存为(A)：
- 总是弹出系统保存对话框。
- 保存成功后，当前文件路径会切换到新保存的文件。

保存时会使用“编码”组合框里选择的编码。
新建歌词默认使用 Unicode UTF-8。

### 12.6 预览编辑框

“预览”编辑框是可编辑的多行编辑框。

你可以在这里：
1. 修改错字。
2. 删除某一行歌词。
3. 调整歌词顺序。
4. 补充作词、作曲等普通文本。

示例：
[00:02.67]哨笛：吴题
[00:12.30]风吹过山岗
[00:18.40]旧梦还在远方

### 12.7 注意事项

- 当前不会在保存后自动绑定为当前朗读歌词。
- 如果想让保存后的歌词被当前歌曲自动识别，建议保存到歌曲同目录、设置的 LRC 歌词目录，或按需要手动加载。
- 如果当前没有播放歌曲，当前时间会显示为 `[00:00.00]`。
- 可以用 Esc、Alt+F4 或关闭(C) 按钮关闭窗口。

## 13. 朗读输出方式

组件支持两种朗读输出方式：
1. 使用屏幕阅读器。
2. 使用 SAPI5.1。

默认使用屏幕阅读器。这个方式通过 Tolk 调用屏幕阅读器朗读歌词。
需要把对应位数的 `Tolk.dll` 放在组件目录下的 `tolk` 文件夹里。
如果提示：
无法加载或调用 Tolk.dll。请确认 Tolk.dll 与读屏支持 DLL 已放在组件同一目录。
请检查：
1. Tolk.dll 是否存在。
2. x64 foobar2000 是否使用 x64 的 Tolk.dll。
3. x86 foobar2000 是否使用 x86 的 Tolk.dll。
4. Tolk.dll 是否放在正确目录。

如果不想使用屏幕阅读器，可以在“朗读歌词设置”的“TTS语音”选项卡里取消勾选“使用屏幕阅读器”，然后选择 SAPI5.1 语音和语速。

注意：组件的 SAPI5.1 朗读是独立实现的。当前组件不使用 Tolk 自带的 SAPI5 托底朗读。

---

## 14. 外部下载器

组件可以调用外部歌词下载器下载歌词。
下载器使用 .NET Framework 4.8 编写。
所以电脑需要安装：
.NET Framework 4.8 运行库
一般 Windows 10/Windows 11 都已经自带或可以很容易安装。
下载器负责从网络来源搜索歌词，并把 `.lrc` 文件保存到临时歌词目录。
组件再从临时目录扫描并加载歌词。

## 15. 常见问题

### 15.1 为什么开头歌词少读了？
可能是“时间偏移（毫秒，负数提前）”设置了负数。
例如：
-1500
如果歌词第一句时间是：
[00:02.67]
提前 1.5 秒后可能太靠前，导致开头歌词处理异常。
解决方法：先改成：
0
再测试。

---

### 15.2 为什么下载歌词后要等一会儿才朗读？

可能是后台扫描时间设置较大。
可以把：
无lrc歌词后台扫描时间毫秒
设置成较小的值，例如：
500
表示 0.5 秒扫描一次。

---

### 15.3 为什么有些 LRC 是这种格式？

例如：
<00:26.223>瓶<00:26.467>身<00:26.736>描<00:27.071>绘
这是逐字时间戳歌词，也叫逐字歌词/卡拉 OK 歌词。
组件主要按普通 LRC 行时间戳朗读，如果来源返回逐字歌词，显示和处理效果可能和普通 LRC 不完全一样。

---

### 15.4 为什么手动加载的歌词会一直用于其他歌曲？

手动加载歌词一般用于临时测试。
如果切换歌曲后仍然用同一个手动歌词，建议改用：
设置LRC歌词目录
或：
设置LRC歌词临时目录
让组件按当前歌曲自动匹配歌词。

---

### 15.5 全部下载来源都不选会怎样？

全部不选时，组件不会调用下载器，只扫描本地歌词目录、临时目录和歌曲同目录歌词。
适合只想用本地歌词的用户。

---

## 16. 推荐配置

如果你主要想自动下载并朗读歌词，推荐这样设置：
自动朗读：开启
LRC歌词编码：自动
时间偏移（毫秒，负数提前）：0
歌词朗读有效时间毫秒：3000
无lrc歌词后台扫描时间毫秒：500
LRC歌词临时目录：%Temp%
下载来源：QQ音乐1号源、QQ音乐2号源、网易云音乐
下载到指定的歌词目录：按需要选择
TTS语音：使用屏幕阅读器保持选中

如果你不想联网下载，只想用本地歌词，推荐这样设置：
自动朗读：开启
LRC歌词编码：自动
LRC歌词目录：你的歌词目录
LRC歌词临时目录：可以不设置
下载来源：全部取消勾选
下载到指定的歌词目录：不用选中
TTS语音：按需要选择屏幕阅读器或 SAPI5.1

---

## 17. 文件命名建议

为了提高匹配成功率，建议 LRC 文件名使用以下格式：
歌手 - 歌名.lrc
例如：
刘珂矣 - 半壶纱.lrc
周杰伦 - 青花瓷.lrc
如果文件名里包含特殊空格、全角空格、特殊符号，可能会影响匹配。
