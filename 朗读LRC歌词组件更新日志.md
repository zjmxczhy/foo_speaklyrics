# 朗读 LRC 歌词组件更新日志

## 0.3.0-2026-06-23

### 新增：添加当前时间歌词
在“视图 -> 朗读歌词”菜单中新增“添加当前时间歌词”功能。  
该功能用于手动制作或补全 LRC 歌词，打开窗口后会自动读取当前播放位置并填入 LRC 时间戳。

窗口包含“当前时间”“当前时间歌词”“编码”“保存”“另存为”“打开”“预览”“关闭”等控件。  
在“当前时间歌词”编辑框中输入歌词后按 Enter，会把当前时间和歌词追加到预览内容中，并自动刷新当前播放时间。

### 新增：LRC 编辑保存能力
新增 LRC 文本打开、保存和另存为流程。  
打开已有 LRC 时会自动识别编码，保存时会按窗口中选择的编码写入文件。

### 调整：编码选项补充 UTF-8 with BOM
LRC 编码列表新增“Unicode (UTF-8 with BOM)”选项。  
新建歌词默认使用 Unicode UTF-8；如果需要兼容某些旧软件，可手动选择 UTF-8 with BOM。

### 说明：保存后不自动绑定当前歌词
“添加当前时间歌词”保存后不会自动绑定为当前朗读歌词。  
如果要让组件自动识别保存后的歌词，建议保存到歌曲同目录、设置的 LRC 歌词目录，或手动加载保存后的 LRC 文件。

## 0.2.2-2026-06-21

### 新增：关于标签页
在“朗读歌词设置”窗口最后新增“关于”标签页。  
标签页中包含“主要用途”只读的编辑框，说明组件用途。
同时新增了查看GitHub仓库，前往GitHub发布页，备用gitee发布页。
点击按钮会使用系统默认浏览器打开对应页面。

## 0.2.1-2026-06-20

### 修复：UTF-8 无 BOM 歌词自动识别误判
修复部分 UTF-8 无 BOM 的 LRC 歌词在“LRC歌词编码”设置为“自动”时，被误判为 Shift-JIS 等其他编码的问题。
感谢禹涵老师发现的bug。

## 0.2.0-2026-06-19

### 新增：TTS 语音设置
在“朗读歌词设置”窗口中新增“TTS语音”标签页。

### 新增：SAPI5.1 语音朗读
TTS 语音页中默认勾选“使用屏幕阅读器”。  
勾选时继续使用 Tolk 调用屏幕阅读器朗读歌词。  
取消勾选后显示 SAPI5.1 相关设置，可选择 Windows 已安装的 SAPI5.1 语音，并可调整语速和试听。

### 新增：本地歌曲同目录 LRC 自动识别
播放本地歌曲时，组件会优先在歌曲文件所在目录查找同名 LRC 歌词。

### 新增：下载到指定的歌词目录
在“朗读歌词设置”的“LRC歌词”标签页中新增“下载到指定的歌词目录”复选框。  
默认未选中，歌词自动下载后保存到 LRC 歌词临时目录。  
选中后，歌词自动下载和搜索歌词自动匹配下载会保存到用户设置的 LRC 歌词目录，适合长期保存歌词文件。

### 调整：LRC 查找顺序
当前自动查找顺序为：
1. 手动导入且绑定到当前歌曲的 LRC。
2. 本地歌曲同目录下的同文件名 LRC，例如 `青花瓷.lrc`。
3. 本地歌曲同目录下的 `艺术家 - 歌名.lrc`。
4. 本地歌曲同目录下的 `歌名 - 艺术家.lrc`。
5. 本地歌曲同目录下按歌名、艺术家做模糊匹配。
6. 用户设置的 LRC 歌词目录。
7. LRC 歌词临时目录。
8. 如果仍未找到，并且启用了歌词来源，则调用下载器从网络来源下载。

### 说明：网络歌曲不扫描同目录
网络来源的歌曲没有本地文件目录，因此不会执行“同目录 LRC 查找”。
网络歌曲会跳过同目录判断，继续检查设置的 LRC 歌词目录、临时目录，然后再按设置的来源下载。

### 调整：设置窗口布局
在“朗读歌词设置”窗口新增“常规设置”标签页。  
当前标签页顺序调整为：`常规设置`、`LRC歌词`、`TTS语音`。

以下控件已移动到“常规设置”标签页：
1. 自动朗读。
2. LRC歌词编码。
3. 时间偏移（毫秒，负数提前）。
4. 无lrc歌词后台扫描时间毫秒。
5. 歌词朗读有效时间毫秒。

### 调整：LRC歌词标签页焦点顺序
“LRC歌词”标签页的焦点顺序调整为：
1. 歌词下载来源。
2. 下载到指定的歌词目录。
3. LRC文件。
4. 浏览文件。
5. LRC歌词目录。
6. 浏览文件夹。
7. LRC歌词临时目录。
8. 浏览文件夹。
9. 切换歌曲删除临时lrc时间毫秒。
